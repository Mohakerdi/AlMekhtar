## 🧭 What is Al-Mekhtar?

<img align="right" width="300" src="https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExcHUxcnc1dHc5ZGI4ZGo1cXhsMmRkd3F2bzE4dGppOW9scjV0ZmtoNiZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/f3iwJFOVOwuy7K6FFw/giphy.gif" />

**Al-Mekhtar** (المختار — *The Chosen One*) is a full-featured rental property platform built as a university graduation project. It bridges the gap between property owners and renters through a smooth, native mobile experience.

- 🏗️ Built from the ground up with **Flutter** for true cross-platform support
- 🌐 Fully localized in **Arabic** and **English** — right-to-left layout included
- 🎨 Adaptive **Dark & Light** themes with beautiful Material Design 3 UI
- ⚡ State-of-the-art **BLoC architecture** for scalable, testable code
- 🔐 Secure **JWT-based authentication** with token management

<br clear="right"/>

## ✨ Features

<div align="center">

| &nbsp; | Feature | Description |
| :---: | :--- | :--- |
| 🔍 | **Property Discovery** | Browse listings with smart filters — price, location, type, and more |
| 💬 | **Real-Time Chat** | Communicate directly between owners and renters |
| ❤️ | **Favorites** | Save and revisit your most-loved listings at any time |
| 📅 | **Bookings** | Full booking flow — request, confirm, and track status |
| 🏘️ | **My Rentals** | Owners can add, edit, and manage their properties |
| 🔔 | **Push Notifications** | Stay updated with instant alerts on activity |
| 👤 | **Authentication** | Secure sign-up / login with JWT and persistent sessions |
| 🌙 | **Adaptive Theming** | Automatic Dark ↔ Light mode — follows system preference |
| 🌍 | **Full Localization** | Complete AR 🇸🇦 and EN 🇬🇧 support with RTL layout |
| 🖼️ | **Image Gallery** | Pinch-to-zoom property photos with smooth carousel |
| 📡 | **Offline Awareness** | Graceful no-internet screen with retry functionality |

</div>

## 🚀 Getting Started

<details>
<summary><b>📋 Prerequisites</b></summary>
<br/>

- [Flutter SDK](https://flutter.dev/docs/get-started/install) — Dart `^3.8.1`
- Android Studio **or** VS Code with the Flutter & Dart plugins
- Android SDK **API 30+** or an iOS Simulator / physical device

</details>

<details open>
<summary><b>⚡ Quick Setup</b></summary>
<br/>

```bash
# 1. Clone the repo
git clone https://github.com/Mohakerdi/AlMekhtar.git
cd AlMekhtar

# 2. Install all dependencies
flutter pub get

# 3. Launch the app
flutter run
```

> 💡 Tip: Run `flutter devices` first to list connected devices and emulators.

</details>

<details>
<summary><b>🏗️ Build for Release</b></summary>
<br/>

```bash
# Android APK
flutter build apk --release

# Android App Bundle
flutter build appbundle --release

# iOS (requires macOS + Xcode)
flutter build ios --release
```

</details>
